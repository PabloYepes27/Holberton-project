Markup language that uses the structure to make sense of web content, for example it defines paragraphs, headers, tables, images and videos on the page.

Cascading rule language that we use to style our HTML content, for example by placing background colors, fonts and marginalizing our content into multiple columns.

Programming language that lets you create new and dynamic content, control media files, create animated images, and more.

PYTHON

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics. Its high-level built in data structures, combined with dynamic typing and dynamic binding, make it very attractive for Rapid Application Development, as well as for use as a scripting or glue language to connect existing components together.

DJANGO

Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design. Built by experienced developers, it takes care of much of the hassle of web development, so you can focus on writting your app without needing to reinvent the wheel. It's free and open source.
